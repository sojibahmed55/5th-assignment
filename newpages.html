Question-1: What are the different ways to select an element in the DOM?

gThe different ways to select an element in the DOM are getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), querySelectorAll(), childNodes, closest(), and getElementsByName().

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML includes HTML tags, innerText includes visible text only, and textContent includes all text, excluding HTML tags.

Question-3: What is event delegation in the DOM?

Event delegation is the practice of attaching a single event listener to a parent element to manage events for child elements via event bubbling.

Question-4:What is event bubbling in the DOM?

Event bubbling is when an event propagates from the target element to its parent elements.

Question-5: How do you create, add, and remove elements using JavaScript?

To create an element, use document.createElement('tagName'). To add the element to the DOM, use parentElement. appendChild(newElement). To remove an element, use element.remove().
